
# Sutra Hosting

Modern Discord bot hosting panel inspired by premium cloud dashboards.

## Features
- Sutra Hosting + Sutra Development branding
- Discord OAuth ready
- Litecoin payment flow support
- Modern immersive dashboard
- Deploy bot UI
- React + Vite frontend
- Express backend starter

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```

## Run Backend
```bash
cd backend
npm install
npm start
```
